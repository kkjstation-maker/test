# 🧬 에겐 vs 테토 성향 진단기 (Egen vs Teto Test)

> **"당신은 공감의 에겐(Egen)인가요, 추진력의 테토(Teto)인가요?"** > 호르몬 기질에 따른 심리 상태를 7가지 질문으로 분석하는 인터랙티브 웹 진단기입니다.

---

### 📋 프로젝트 소개
이 프로젝트는 에스트로겐(Estrogen)과 테스토스테론(Testosterone)의 앞 글자를 딴 신조어 **'에겐'**과 **'테토'**를 기반으로 한 성향 테스트입니다. 단순히 성별을 나누는 것이 아니라, 개인의 심리적 기질을 현대적인 관점에서 분석합니다.

### ✨ 주요 기능
* **7문항 정밀 진단:** 심리학적 근거를 바탕으로 설계된 질문.
* **5단계 답변 시스템:** 세밀한 답변 선택 가능.
* **반응형 디자인:** 모바일에서도 최적화된 UI 제공.

### 🧪 유형 안내

| 유형 | 호르몬 기반 | 핵심 키워드 |
| :--- | :--- | :--- |
| **에겐 (Egen)** | 에스트로겐 | 공감, 섬세함, 다정함, 관계 지향 |
| **테토 (Teto)** | 테스토스테론 | 추진력, 논리, 결단력, 목표 지향 |
| **밸런스 (Balance)** | 중립 | 유연함, 상황 적응력, 균형 감각 |
